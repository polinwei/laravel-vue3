## 如何執行測試 Running the tests

<p>執行下列指令安裝</p>
<p>Execute the following instructions to install</p>

安裝 laravel 相關套件
```
composer install
```
執行 Laravel 服務
```
php artisan serve 
``` 
安裝 npm 套件
``` 
npm install
``` 
執行 vite
``` 
npm run dev
或
npm run build
```
